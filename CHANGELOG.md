# Changelog

## 0.1.0
- Added automatic brightness detection using viewport grid sampling
- Added configurable dim intensity (0-100%)
- Added configurable brightness threshold for triggering dimming
- Added dynamic mode that scales dimming based on page brightness
- Added per-site whitelist for always dimming specific sites
- Added per-site blacklist to exclude sites from dimming
- Added smooth transitions with configurable duration
- Added settings popup with real-time controls
- Added Chrome storage sync for cross-device settings
- Added per-site state persistence to prevent flash on page load
- Added exponential brightness smoothing for stability
- Added change thresholds to prevent flickering
- Added comprehensive unit tests for all utility modules
- Added Puppeteer E2E tests for extension functionality
